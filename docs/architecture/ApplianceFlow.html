Destinations

  • Internet

Key Points

  • Traffic from the internal network is routed through the GSA Appliance.
  • Global Secure Access provides secure connectivity to Internet and Microsoft 365.
  • The GSA Appliance establishes an IPsec tunnel with Global Secure Access.
  • Public IPs are used for external connectivity and remote network access.
  • BGP is used for dynamic routing between the GSA Appliance and Global Secure Access.
  • Microsoft 365 and Internet traffic are tunneled, then distributed at the GSA edge.
